# LazyTables v0.2.2 Release Notes

**Release Date:** October 12, 2025

**Release Type:** Stable Release - Bug Fixes & Enhancements

---

## Overview

Version 0.2.2 brings critical bug fixes, enhanced vim-style navigation, and expanded database support. This release focuses on improving reliability and user experience with refined documentation.

## What's New

### 🐛 Critical Bug Fixes

- **SQLite Connection Fix** - Corrected SQLite connection URL format (`sqlite://` instead of `sqlite:`), resolving error code 14 that prevented app_state.db creation
- **Database Adapter Security** - Enhanced SQL injection protection and error handling across MySQL, MariaDB, and SQLite adapters

### ✨ Navigation Enhancements

- **Page Scrolling** - Added `Ctrl+d` (page down) and `Ctrl+u` (page up) in table viewer
- **Jump Commands** - Implemented `gg` (jump to top) and `G` (jump to bottom) for both Data and Schema views
- **Tab Management** - Added `x` key to close individual table tabs with confirmation toast

### 🗄️ Database Support

- **MySQL & MariaDB** - Full adapter support with connection pooling and secure credential handling
- **Enhanced SQLite** - Improved identifier validation and error formatting
- **Schema Viewing** - Comprehensive metadata display for columns, indexes, foreign keys, and constraints

### 📚 Documentation

- **Streamlined README** - Condensed from 454 to 151 lines with clearer structure and visual separators
- **Focused Guides** - Removed verbose content, emphasizing quick start and essential workflows
- **Release Notes** - Updated to reflect current stable features

## Installation

### From crates.io

```bash
cargo install lazytables
```

### Using cargo-binstall

```bash
cargo binstall lazytables
```

### From Source

```bash
git clone https://github.com/yuyudhan/LazyTables.git
cd LazyTables
cargo install --path .
```
