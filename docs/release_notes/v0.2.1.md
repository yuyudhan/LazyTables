# LazyTables v0.2.1 Release Notes

**Release Date:** October 11, 2025

**Release Type:** Stable Release

---

## Overview

LazyTables v0.2.1 marks the first stable release of our major async architecture overhaul. This release brings significant performance improvements, enhanced user experience, and a fully non-blocking interface that ensures smooth and responsive database interactions.

## What's New

### 🚀 Performance & Architecture

- **Fully Async Architecture** - All database and file operations are now non-blocking, providing a smooth and responsive experience even with large datasets or slow database connections
- **Non-blocking Database Connections** - Connect to databases without freezing the UI, with visual feedback and animated loading indicators
- **Async File I/O** - All file operations now use async I/O with timeout protection (5-second default)

### ✨ User Experience Enhancements

- **Vim-style Command Mode** - SQL query editor now supports vim commands (`:w` to save, `:q` to quit, `:wq` to save and quit)
- **SQL Autocomplete** - Intelligent keyword suggestions and context-aware completions in the SQL editor
- **Numbered Pane Navigation** - Press 1-6 to jump directly to any pane for faster navigation
- **Tab Navigation in Table Viewer** - Use Shift+H/L to quickly switch between open tables
- **Collapsible Table Browser** - Better organization with collapsible groups for schemas and table types
- **Animated Loading Indicators** - Visual feedback with timer display during connection attempts and long-running operations

### 🐛 Bug Fixes

- Fixed Tab key behavior in SQL editor insert mode (no longer switches panes during text editing)
- Resolved async runtime panics in database and file operations
- Improved connection testing UI responsiveness
- Fixed table navigation and scrolling edge cases
- Eliminated all clippy warnings for cleaner, more maintainable code

### 🛠️ Technical Improvements

- Unified view hierarchy system for better state management
- Event-driven overlay routing for improved modal and overlay handling
- Enhanced connection modal UX with step-by-step guidance
- Updated help system with numbered navigation and new features
- Comprehensive timeout protection for all I/O operations
- Improved error handling with better user feedback

## Installation

### From crates.io

```bash
cargo install lazytables
```

### Using cargo-binstall (Fast Binary Installation)

```bash
cargo binstall lazytables
```

### From Source

```bash
git clone https://github.com/yuyudhan/LazyTables.git
cd LazyTables
cargo install --path .
```

## Upgrade Notes

If you're upgrading from v0.1.x, your existing connections and configuration will be preserved. The async architecture changes are transparent to existing configurations.

## Known Limitations

- Windows platform is not officially supported (macOS and Linux only)
- Oracle, MongoDB, and other database types are planned for future releases (currently supports PostgreSQL, MySQL, SQLite)

## What's Next

- Enhanced query history and favorites
- Advanced filtering and search capabilities
- Export functionality for query results
