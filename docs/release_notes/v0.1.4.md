# Release v0.1.4 - Debug View & Connection Management

**Release Date:** 2025-09-24

---

## Overview

LazyTables v0.1.4 introduces essential debugging capabilities and improves connection management. The highlight of this release is the full-screen debug view that helps users troubleshoot issues in real-time.

## ✨ New Features

### Debug View
- **Full-screen debug overlay** activated with Ctrl+B (changed from Ctrl+D to avoid conflicts)
- **Real-time logging system** visible during application use
- Live log tail showing latest debugging information
- Structured logging with levels (DEBUG, INFO, WARN, ERROR)
- Toggle debug view on/off without losing state

### Connection Management
- **Persistent connection state** across sessions
- Improved connection lifecycle management
- Better error handling for connection failures
- Connection status indicators in UI

### Data Editing
- **Database cell editing capabilities** (initial implementation)
- In-cell editing with keyboard navigation
- Save changes back to database
- Error handling for update failures

## 🎨 UI Improvements

### Dark Theme
- Fixed dark theme color inconsistencies
- Better contrast for focused elements
- Improved visibility of status indicators
- Consistent color scheme across all panes

### Debug View State
- Proper state management for debug overlay
- Non-intrusive display over main UI
- Keyboard shortcuts documented in help system
- Smooth toggle animation

## 🔧 Technical Improvements

### Logging Infrastructure
- Structured logging with tracing crate
- Log file rotation to prevent disk space issues
- Configurable log levels (via environment)
- Performance-optimized log collection

### State Management
- Better separation of UI and database state
- Improved state synchronization
- Reduced state-related bugs
- More predictable behavior

## 🐛 Bug Fixes

- Fixed Ctrl+D conflict (moved debug to Ctrl+B)
- Improved connection error messages
- Fixed state persistence issues
- Better handling of edge cases in table navigation

## 📦 Installation

```bash
# From crates.io
cargo install lazytables

# From source
git clone https://github.com/yuyudhan/LazyTables.git
cd LazyTables
cargo install --path .
```
