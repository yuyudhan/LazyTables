# Release v0.1.5 - Version Bump

**Release Date:** 2025-09-26

---

## Overview

LazyTables v0.1.5 is a technical release focused on testing the cargo-binstall integration. No functional changes were made in this version.

## 🔧 Technical Changes

### cargo-binstall Testing
- Version bump to test cargo-binstall workflow
- Validation of binary packaging process
- Testing of automated release pipeline
- No functional or feature changes

## Purpose

This release served as a testing ground for the improved installation process that would be fully implemented in v0.1.7. The primary goal was to ensure our CI/CD pipeline could properly generate binaries compatible with cargo-binstall.

## Installation

```bash
# Standard cargo installation (recommended for this version)
cargo install lazytables

# From source
git clone https://github.com/yuyudhan/LazyTables.git
cd LazyTables
git checkout v0.1.5
cargo install --path .
```

## What Happened After

The testing conducted in v0.1.5 led to:
- Successful cargo-binstall integration in v0.1.7
- Improved binary packaging workflow
- Faster installation for end users
- Multi-platform pre-built binary support

---

**Note**: Users should upgrade directly to v0.1.6 or later for functional improvements. This version exists primarily for development pipeline validation.
