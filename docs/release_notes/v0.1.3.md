# Release v0.1.3 - First Stable Release

**Release Date:** 2025-08-15

---

## Overview

LazyTables v0.1.3 marks the first stable release of the terminal-based SQL database viewer and editor. This release establishes the foundation with secure credential management, SQL file organization, and basic database operations.

## ✨ Features

### Secure Password Management
- **AES-256-GCM encryption** for database credentials
- **Argon2 key derivation** for password protection
- Support for **environment variables** in connection strings
- Encrypted password storage in `~/.lazytables/connections.json`

### SQL File Management
- **Per-database organization** of SQL files
- Connection-specific SQL file directories
- File browser for saved queries
- Load and execute saved SQL queries

### Query Execution
- **Execute SQL under cursor** with Ctrl+Enter
- Query results displayed in table viewer tabs
- Support for multiple result sets
- Basic error handling and display

### Connection Management
- Single active connection enforcement
- Connection deletion with confirmation modal
- Connection state persistence
- Database type icons (🐘 PostgreSQL)

## 🔧 Improvements

### User Experience
- Removed vim-style insert mode from connection modal
- Updated help text for all panes with accurate keybindings
- Added confirmation modals for destructive actions
- Cleaner, more intuitive connection workflow

### Code Quality
- **Zero clippy warnings** - all linting issues resolved
- Proper code formatting with rustfmt
- Improved error messages and user feedback
- Better state management

## 🗄️ Database Support

### Supported
- **PostgreSQL** - Full support with connection pooling

### Planned
- MySQL/MariaDB
- SQLite
- Oracle
- MongoDB
- Redis

## 🛠️ Technical Stack

- **Language**: Rust 1.70+
- **TUI Framework**: Ratatui
- **Async Runtime**: Tokio
- **Database**: SQLx (PostgreSQL)
- **Encryption**: AES-GCM + Argon2
- **Platforms**: macOS, Linux

## 📦 Installation

```bash
# From source
git clone https://github.com/yuyudhan/LazyTables.git
cd LazyTables
cargo install --path .
```

## 🐛 Known Issues

- Only PostgreSQL supported in this release
- Limited query result formatting
- No horizontal scrolling for wide tables
- Connection testing can block UI briefly
